# snap-view

It is a lightweight JavaScript library that allows you to capture screenshots of the visible part of your web page effortlessly

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```
